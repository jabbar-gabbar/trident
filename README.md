# trident
Utility to upload and sync local folders to aws s3 bucket.
Supported on Windows environment.

### Pre-requisite
1. Visual Studio 2017 Community (free!)

2. .NET Framework 4.6 or older

3. Windows 7 or older

4. AWS account
